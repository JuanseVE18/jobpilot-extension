# 🚀 JobPilot – Smart Job Application Autofill Extension

JobPilot is a Chrome Extension that helps users autofill job applications quickly using a saved professional profile.

## ✨ Features

- Save personal information (name, email, phone, LinkedIn)
- Store experience, education, projects, and certifications
- Autofill job application forms with one click
- Clean and minimal user interface
- Data stored securely using Chrome local storage

## 🛠 Tech Stack

- JavaScript (Vanilla)
- Chrome Extensions API (Manifest V3)
- HTML5
- CSS3
- Chrome Storage API

## 📂 Project Structure

```
extension/
│
├── popup/          # Extension UI logic
├── content/        # Autofill logic injected into pages
├── shared/         # Shared helpers and schema
├── icons/          # Extension icons
├── manifest.json   # Chrome extension configuration
```

## 🚀 How to Install Locally

1. Clone the repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer Mode"
4. Click "Load unpacked"
5. Select the `extension` folder
